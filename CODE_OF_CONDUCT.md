# Contributor Code of Conduct

## Our Pledge

We are committed to providing a respectful and harassment-free environment for
everyone contributing to `ide-super-worker`.

## Standards

- Be respectful in discussions, reviews, and issues.
- Focus on technical merit and evidence.
- Accept constructive feedback gracefully.
- Prioritize correctness and security in all contributions.

## Unacceptable Behavior

- Harassment, personal attacks, or discriminatory language.
- Publishing others' private information without consent.
- Claiming others' work as your own (plagiarism).
- Submitting code that violates another project's license.

## Enforcement

Violations may result in issue closure, PR rejection, or being blocked from the
repository. Report incidents via GitHub private security advisory or the issue
tracker with a "conduct" label.
